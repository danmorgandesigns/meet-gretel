Professional Trail Development

Create detailed trail maps with GPS precision for new trail construction, existing trail documentation, and maintenance planning. Record exact routes with customizable accuracy settings to meet surveying standards. Automatically calculate trail length and elevation profiles essential for accessibility compliance and difficulty ratings.

Route Precision and Elevation Smoothing

Using the Route Precision slider, the User can customize the minimum distance between saved points as they track a route. The Update Frequency slider changes the what the minimum movement needs to be before a new location update is recorded while tracking.

Smaller values results in larger route tracking files, but make the route more detailed with smoother curves. Larger values create smaller files, but the route may appear more angular with sharp turns.

Unit Conversions

Gretel allows users to view trail distances and elevations in either Metric or Imperial units. This flexibility ensures that professionals worldwide can work with their preferred measurement systems, enhancing usability and accuracy in field mapping tasks.
